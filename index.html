Lenguage used

HTML

Hypertext Markup Language (HTML) é o código usado para estruturar e exibir uma página da web e seu conteúdo. Por exemplo, seu conteúdo pode ser parágrafos, uma lista com marcadores ou imagens e tabelas de dados. Como o título sugere, este artigo lhe dará uma compreensão básica do HTML e do que ele faz.

CSS

CSS (Cascading Style Sheets) é o código que você usa para estilizar seu site. CSS básico orienta você sobre o que você precisa para começar. Ele responderá a perguntas como: Como faço para deixar meu texto vermelho ou preto? Como faço para que meu conteúdo seja exibido em tal e tal lugar da tela? Como posso decorar meu site com imagens e cores de fundo? Então, o que é CSS, realmente? Assim como HTML, CSS (Cascading Style Sheets) ou Cascading Style Sheets em espanhol, não é realmente uma linguagem de programação, nem é uma linguagem de marcação. É uma linguagem de folha de estilos, ou seja, permite aplicar estilos seletivamente a elementos em documentos HTML.

JavaScript

JavaScript (JS) é uma linguagem de programação leve, interpretada ou compilada justa-a-tempo (just-in-time) com funções de primeira classe. Embora seja mais conhecido como uma linguagem de script (sequências de comandos) para páginas da web, e é usado em muitos ambientes fora do navegador, como Node.js, Apache CouchDB e Adobe Acrobat JavaScript é uma linguagem de programação baseada em protótipos, multiparadigma, de um único hilo, dinâmico, com suporte para programação orientada a objetos, imperativa e declarativa (por exemplo, programação funcional). O padrão para JavaScript é ECMAScript (ECMA-262) e a especificação da API para internacionalização de ECMAScript (ECMA-402). A documentação no MDN é baseada profundamente nas últimas versões preliminares de ECMA-262 e ECMA-402. E em alguns casos onde algumas propostas para novas funções para ECMAScript já foram descobertas nos navegadores, a documentação e alguns artigos do MDN podem fazer com que algumas dessas funções sejam usadas. Não confunda JavaScript com a linguagem de programação Java. Ambos "Java" e "JavaScript" são marcas ou marcas registradas da Oracle nos Estados Unidos e em outros países. No entanto, os dois idiomas de programação têm sintaxe, semântica e usos muito diferentes.
